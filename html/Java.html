Java je objektově orientovaný programovací jazyk vytvořený společností Sun Microsystems. Je velmi populární díky tomu, že programy napsané v Javě mohou fungovat na různých operačních systémech. Java se používá při tvorbě podnikových aplikací, webových systémů a mobilních aplikací pro Android. Jazyk je známý svou stabilitou a bezpečností.
