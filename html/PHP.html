PHP

Domů Python Java C# C++ PHP

PHP je skriptovací programovací jazyk používaný hlavně pro tvorbu dynamických webových stránek. PHP pracuje na straně serveru a často se používá společně s databázemi. Pomocí PHP lze vytvářet přihlašovací systémy, formuláře nebo celé webové aplikace. Mnoho známých webů bylo vytvořeno právě pomocí tohoto jazyka.
