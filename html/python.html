Python je populární programovací jazyk, který vznikl v roce 1991. Používá se pro tvorbu webových aplikací, práci s databázemi, umělou inteligenci, analýzu dat i automatizaci úloh. Python je známý svou jednoduchou syntaxí, díky které je vhodný pro začátečníky. Mnoho firem používá Python například pro vývoj moderních aplikací nebo strojového učení.
